# ShortCircuit — Venture Studio

Landing page for [shortcircuit.ventures](https://shortcircuit.ventures)

## Stack
Single-file HTML — Space Mono + Inter via Google Fonts, vanilla CSS, no dependencies.

## Files
- `index.html` — Landing page
- `brand.html` — Brand identity sheet (wordmark, palette, type specimen)

## Deploy
Drop `index.html` anywhere that serves static files (Cloudflare Pages, Vercel, Netlify).
